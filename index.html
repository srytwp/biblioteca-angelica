Uma biblioteca feita para parecer um refúgio: música clássica, memória, silêncio bonito e livros escolhidos com carinho.

Use fones de ouvido

Essa experiência foi pensada primeiro para celular. Coloque os fones, toque a música e abra os livros como se estivesse entrando em uma sala secreta feita só para você.

Abrir no YouTube

Obs: alguns celulares bloqueiam música automática. Por isso ela começa quando você toca no botão.

Livros da biblioteca

Para livros ficarem disponíveis para a Angélica, envie o PDF no GitHub e crie um livro fixo no código. O botão “Adicionar PDF local” abre apenas no aparelho de quem adicionou.
